# TCP_demo

#### 介绍

计网课程的三级项目

##### 题目

项目2：发送和接收TCP数据包

(a) TCP数据包结构设计；

(b) TCP数据包发送和接收过程。

TCP是一种面向连接的、可靠的传输层协议。TCP协议工作在网络层IP协议的基础上。本项目的目的是设计一个发送和接收TCP数据包的程序，其功能是填充一个TCP数据包，发送给目的主机，并在目的主机接收此TCP数据包，将数据字段显示显示在标准输出上。

#### 软件架构

- TcpSender下是发送方源码
- TcpReceiver下是接收方源码
- com-headers下是公用头文件和API


#### 开发环境

- Visual Studio 2022
- Npcap 1.73
- Npcap 1.13 SDK

1.  自行安装和配置Npcap
